# Шило на мыло 🧼🕯️
**Тема:** Сайт мастерской по изготовлению свечей и мыла ручной работы  
**Название проекта:** «Шило на мыло»  
**Слоган:** Меняем суету на чистоту
---
## 📖 О проекте
Этот проект — учебная разработка полноценного веб-сайта для небольшой хендмейд-мастерской. Концепция построена на ироничном обыгрывании поговорки: сайт предлагает клиентам в прямом смысле «обменять шило на мыло» — отдать накопившийся стресс и получить взамен натуральные продукты для уюта и заботы о себе.
Сайт задуман не просто как витрина товаров, а как место с характером и атмосферой: здесь важны юмор, теплота ручной работы и прозрачность состава.
---
## 🎯 Целевая аудитория
Люди, которые ценят натуральные составы, уют и самоиронию. Те, кто покупает мыло не просто чтобы мыться, а чтобы смыть с себя суету — и не прочь улыбнуться в процессе.
