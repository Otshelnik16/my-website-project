---
## Пошаговая инструкция
### 1. Установка плагина bbPress
- Админка → Плагины → Добавить новый → поиск "bbPress" → Установить → Активировать.
---
### 2. Настройка постоянных ссылок
- Настройки → Постоянные ссылки → Сохранить изменения (для обновления ЧПУ).
---
### 3. Создание форума
- Форумы → Добавить новый.
- Название: "Общие обсуждения".
- Описание: "Форум для обсуждения наших продуктов, обмена опытом и общения с единомышленниками".
- Тип: Форум.
- Опубликовать.
---
### 4. Добавление форума в меню
- Внешний вид → Меню.
- Блок "Форумы" → отметить "Общие обсуждения" → Добавить в меню.
- Сохранить меню.
